## Framework

Lensed Build is a double-diamond process with logic gates.

**Each dimension checked by interrogation:**

- UX dimension: Is the problem real? Is user need clear?
- Business dimension: Is there commercial viability? Is proposition sound?
- Code dimension: Is it feasible? Is it built for change?

**Interrogation states:**

- `UX_INCOMPLETE` → close gap with skill → re-interrogate
- `BUSINESS_INCOMPLETE` → close gap with skill → re-interrogate
- `CODE_INCOMPLETE` → close gap with skill → re-interrogate
- `ALL_COMPLETE` → proceed to phases

**Phases (Discover → Define → Design → Deliver):** Double-diamond applied. Diverge, converge, move to next phase.

AI integrated throughout — accelerate, don't bypass thinking.
